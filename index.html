1.1 什么是CSS

Cascading Style sheet 层叠级联样式表

CSS：表现（美化网页）

字体，颜色，边距，高度，宽度

1.2 发展史

CSS1.0 基本样式

CSS2.0 DIV（块）+CSS，HTML与CSS结构分离的思想，网页变得简单

CSS2.1 浮动，定位

CSS3.0 圆角边框，阴影，动画… 浏览器兼容性~

1.3 快速入门

style

基本入门

建议使用这种规范

CSS优势：

  1. 内容和网页分离

  2. 网页结构表现统一，可以实现复用

  3. 样式十分地丰富

  4. 建议使用独立于html的css文件

  5. 利用SEO，容易被搜索引擎收录！

1.4 CSS的三种导入方式

拓展：外部样式两种写法

  • 链接式:

    html

  • 导入式:

    @import是CSS2.1特有的！

    ​ 首先link和@import语法结构不同，前者<link>是html标签，只能放入html源代码中使用，后者可看作为css样式，作用是引入css样式功能。（所以有些网页网卡时只能加载出很原始的界面）

    ​ 本质上两者使用区别不大，但为了软件中编辑布局网页html代码，一般使用link比较多，也推荐使用link。

2.选择器

作用：选择页面上的某一个或者某一类元素

2.1 基本选择器

  1. 标签选择器：选择一类标签

  2. 类选择器 class：选择所有class属性一致的标签，跨标签 .class{}

  3. id选择器：全局唯一！！ #id{}
